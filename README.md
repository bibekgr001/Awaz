# Awaz - The Voice of Gen Z

Simple social voice-posting prototype (Assignment 3 improvements)

## What changed since Assignment 2
- Improved UI (Material3 top bar, nicer post cards)
- Live feed with likes and immediate post visibility
- Profile screen showing user posts
- Snackbars on login/register for immediate feedback
- Firestore rules secured (must be authenticated)
- Launcher icon added

## How to run
1. Clone repo
2. Open in Android Studio (Kotlin + Compose)
3. Add google-services.json (already in repo)
4. Run on emulator/device
