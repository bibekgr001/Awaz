## Summary
Awaz is a voice-first social app for Gen Z to share short audio thoughts and connect.

## Assignment 3 Upgrades
- Secured Firestore rules so only authenticated users can read/write their data.
- Added a custom launcher icon.
- Polished the Home screen UI (cards + feed) for improved UX.
- Added ability to create (register) and read (display email) data in Firestore.
- Included instructions for building a signed release (AAB/APK).

## How to run
1. Clone the repository.
2. Open in Android Studio.
3. Ensure `google-services.json` is in `app/`.
4. Build and run on emulator or device.

## Secure Firebase
Firestore rules are implemented to allow only authenticated users to read/write their own user documents and to allow only authenticated users to access posts.
